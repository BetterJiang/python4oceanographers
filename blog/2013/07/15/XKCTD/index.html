XKCTD plot fun

Jul 15, 2013

Jakecob Vanderplas Sketch Path Filter made into the core of matplotlib 1.4.x. Now it is was easy to produce plot with the XKCD style with just the following:

In [3]:
import matplotlib.pyplot as plt
plt.xkcd()  #<- That's it!
Out[3]:
<matplotlib.rc_context at 0x4550750>

You will need the latest development version of matplotlib. (If you use openSUSE 12.3 just add my custom repository, here is a list of the packages available).

And here is my XKCTD profile ;-)

In [13]:
fig, ax = downcast['t090c'].plot()
ax.set_title("XKCTD")
ax.set_xlabel("Temperature")
ax.set_ylabel("Pressure [dbar]")
fig.set_size_inches((6, 8))
